Small Directory is built & maintained by Samantha Riegl.

It is built with an updated version of leafy (PapaParse.js & Google sheets).
ChatGPT assisted in making tweaks to the updated code because JavaScript is hard.
The site is hosted on Netlify & the source code is publicly available on Github.

Styling is a combination of CSS Micro Reset & custom CSS.

Text is set using the follwing typefaces originally created by Ange Degheest:
FT88 Regular ❎ FT88 Italic ❎ FT88 School ❎ FT88 Bold
The digital revival of these fonts are distributed for free and in open source by Velvetyne Type Foundry.
